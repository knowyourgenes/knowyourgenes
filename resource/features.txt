9. CROSS-CUTTING / PLATFORM FEATURES
---------------------------------------------------------
Auth & roles
    - Role-gated routes: User, Agent, Admin, Counsellor, Guest
    - Email/Google/OTP login
    - Password reset via email

Payments
    - Razorpay integration (UPI, card, net banking, EMI)
    - Promo-code engine (flat & percent, usage caps, expiry)
    - Receipts + GST-compliant invoices

Notifications (WhatsApp + Email automations via WhatsApp Business API)
    - Order confirmation (immediate)
    - Pre-visit instructions (T-24h)
    - Day-of reminder (T-2h)
    - Phlebotomist en-route ping
    - Sample-received / report-ready alerts
    - Reschedule / cancellation flows
    - 3-day, 30-day, 6-month post-report follow-ups
    - Cart-abandonment recovery (30-min window)

Order lifecycle & tracking
    - Unified state machine (Booked → Agent assigned → En route → Collected → At lab → Report ready / Cancelled)
    - Shiprocket chain-of-custody hand-off
    - Public tracking URL (no auth)

Reporting (clinical)
    - Counsellor mandatory QC before release
    - Critical-finding protocol: counsellor calls before digital delivery
    - Plain-English summary + technical appendix
    - Color-coded Low/Moderate/High indicators
    - PDF export + secure doctor-share links

Phlebotomist operations
    - On-time SLA: auto ₹200 discount if >30 min late
    - No-show flow (15-min wait, auto-reschedule link)
    - Sample-rejection flow (free re-collection in 72 hrs)
    - Digital consent form (tablet/phone, e-signed)
    - Tamper-evident seals + QR-based sample labelling

Privacy & compliance
    - End-to-end encryption at rest + in transit
    - User-initiated data deletion
    - Sample-destruction policy (60 days post-report unless opted in)
    - PCPNDT / ACMG / NABL compliance
    - Research opt-in (anonymous, separate consent)

SEO & marketing
    - XML sitemap (public routes only; prioritised per launch doc)
    - Open Graph / Twitter cards
    - GA4 + Meta Pixel + Google Ads conversion events
    - Exit-intent + cart-abandonment retargeting
    - Structured data (Product, FAQ, Organization)

Performance & UX
    - Server components + streaming (Next.js 16)
    - Prefetched navigation, loading.tsx skeletons for dynamic routes
    - Mobile-first agent UI, responsive dashboards
    - Accessibility: WCAG 2.1 AA targets

Observability
    - Error boundary per segment (error.tsx)
    - Structured logs for order-state transitions
    - Operations dashboard for TAT & SLA monitoring

10. KPIS TO INSTRUMENT FROM DAY ONE
---------------------------------------------------------
Discovery       — bounce < 55%, session > 90s
Booking         — test → checkout > 60%, checkout → pay > 75%, AOV > ₹10,000
Confirmation    — no-show < 5%, WhatsApp open > 85%
Collection      — on-time > 90%, sample rejection < 3%, CSAT > 4.5/5
Report          — <14-day delivery > 95%, clarity rating > 80% "easy"
Consultation    — report → booking > 30%, CSAT > 4.5/5
Retention       — referral > 10%, 6-month re-test > 20%
Overall         — NPS > 70, CAC < ₹2,500
